
# Experiment 04 - Verification of QCNN Behavior via Weighted Model Counting

## Overview
Once the scaled-up experiments are running, we will use Weighted Model Counting (WMC) to verify QCNN classifications. For inputs with known classifications, we apply WMC using Quokka# to compute the probability of measuring the outcome corresponding to the correct QCNN classification.

## Key Objectives
- Verify QCNN classifications using Weighted Model Counting
- Compute probability of correct measurement outcomes for known inputs
- Benchmark verification runtime against established techniques

## Methodology
- Apply WMC to trained QCNN models
- Evaluate probability distributions of classification outcomes
- Compare performance with traditional verification approaches
